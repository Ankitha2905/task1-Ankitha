# 🤖 Rule-Based AI Chatbot

## Overview

This project was developed as part of an Artificial Intelligence learning module. The objective of the project is to create a simple rule-based chatbot that interacts with users through predefined responses. The chatbot processes user input and responds according to programmed rules, demonstrating the fundamentals of conversational AI and human-computer interaction.

The chatbot operates in a command-line environment and can answer common greetings and basic questions. It continues the conversation until the user chooses to exit.

---

## Features

* Interactive command-line chatbot
* Responds to greetings such as Hi, Hello, and Hey
* Answers simple predefined questions
* Continuous conversation loop
* Exit functionality using a specific command
* Easy-to-understand Python implementation

## How It Works

The chatbot follows a rule-based approach:

1. Accepts user input.
2. Converts the input to lowercase.
3. Compares the input against predefined rules.
4. Generates an appropriate response.
5. Continues the conversation until the user enters the exit command.
